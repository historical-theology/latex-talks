\begin{frame}
    \frametitle[Everything]{\LaTeX\ for Everything}
The world's best established software system for typesetting and document formatting, \LaTeX\ has been popular throughout the mathematical and physical sciences for decades. The utility of \LaTeX\ for scholars in humanities disciplines, however, tends to be overlooked. \break

\LaTeX\ can be used for typesetting in most instances in which a scholar of humanities needs to present our materials. \break

I prepared these slides in \LaTeX\ -- with less frustration than would have been possible in graphical presentation program (e.g. LibreOffice Present) -- as 1 file: \break

\small{\href{https://github.com/historical-theology/latex-talks/blob/main/icms/medievalist-latex.tex}{github.com/historical-theology/latex-talks/blog/main/icms/medievalist-latex.tex}}
\end{frame}

\begin{frame}
	\frametitle[Tutorial]{This Tutorial}
In this tutorial, I will introduce attendees to \LaTeX, as well as to the use of \LaTeX\ for a common task in medievalist scholarship, namely, typesetting texts with unusual line length and other organizational needs, such as primary texts with our original translations. \break

The specific `thought experiment' will that of determining how to prepare a handout for a historical conference or class presentation with perfectly organized excerpts of a Byzantine Greek text and an original English translation in parallel. \break
\end{frame}

\begin{frame}
	\frametitle[\LaTeX]{What is \LaTeX ?}
It might be best to start to understand what \LaTeX\ \emph{is} by understanding what it is \emph{not}. \break

A word processor, such as LibreOffice Writer or Microsoft Word, presents material in a (W)hat (Y)ou (S)ee (I)s (W)hat (Y)ou (G)et (WYSIWYG, ``whizzy-wig'') style. While one writes, one is simultaneously thinking about and working on formatting. \break

In a typesetting language, such as \LaTeX\ (or ConTeXt or Typst, among others), one is only concerned about the \emph{structure} of what one is writing. One declares that specific characters will be in certain categories of writing `things' (heading, paragraph, footnote, accent type, typeface, etc.), and one allows the program to compile one's writing as `code' into a tidy final document -- every time.
\end{frame}

There are many ways to set parallel text in \LaTeX. \break

To start using \texttt{parallel} in any \LaTeX\ document, simply include the following in the preamble: \break
 
\verb|\usepackage{parallel}| \break

\begin{frame}[fragile]
	\frametitle[Using parallel]{Using the \texttt{parallel} package}

Basic Format

{\scriptsize \begin{lstlisting}
\begin{Parallel}{}<left adjustment>}{<right adjustment>}
		\ParallelLText{<left text row 1>}
		\ParallelRText{<right text row 1>}
			\ParallelPar
		\ParallelLText{<left text row 2>}
	...	
\end{Parallel}	
\end{lstlisting}}

Example
	
{\scriptsize \begin{lstlisting}
\begin{Parallel}{0mm}{0mm}
	\ParallelLText{Ave Maria,}
	\ParallelRText{Hail Mary,}
		\ParallelPar
	\ParallelLText{gratia plena,}
	\ParallelRText{full of grace,}
\end{Parallel}	
		\end{lstlisting}}

\begin{Parallel}{0mm}{0mm}
\texttt{\ParallelLText{Ave Maria,}
\ParallelRText{Hail Mary,}
\ParallelPar
\ParallelLText{gratia plena,}
\ParallelRText{full of grace,}
}\end{Parallel}	

\end{frame}

\begin{frame}[fragile]
	\frametitle[Advanced Parallel Text]{Advanced Setting of Parallel Text in \LaTeX}
	
For \emph{advanced} scholarly setting of parallel text in \LaTeX, one ought to learn the 2 interconnected and frequently updated community packages \texttt{reledpar} and \texttt{reledmac}, which are bundled with TeXLive and other \LaTeX\ distributions. \break

These 2 packages are specifically designed for long-form academic parallel texts, and they are meant to replace older methods of academic parallel text preparation in \LaTeX\ that you might find in Web searches. \break

For full documentation, visit: \break
\href{https://ctan.org/pkg/reledpar}{ctan.org/pkg/reledpar} \break
\href{https://ctan.org/pkg/reledmac}{ctan.org/pkg/reledmac} \break

Remember them in the preamble to your document:

\begin{lstlisting}
\usepackage{reledpar}
\usepackage{reledmac}
\end{lstlisting}

\end{frame}

\begin{frame}[fragile]
	\frametitle[Non-Latin Text]{Setting of Non-Latin Text in \LaTeX}

The most important package for handling text outside of Latin alphabets is the community package \texttt{polyglossia}, which is (as you probably have guessed) included in TeXLive and other \LaTeX\ distributions. \break

This continuously updated package is meant to replace older methods of non-Latin text entry that you might find in Web searches. \break

For full documentation, visit: \break
\href{https://www.ctan.org/pkg/polyglossia}{ctan.org/pkg/polyglossia} \break

\end{frame}

\begin{frame}[fragile]
	\frametitle[Using polyglossia]{Starting a Project with \texttt{polyglossia}}

Remember the package in the preamble:

\begin{lstlisting}
\usepackage{polyglossia}
\end{lstlisting}

Then, \emph{after} the beginning of the document but \emph{before} actual type, declare the languages and their variants. For example, here is this .tex file's polyglossia declarations:

\begin{lstlisting}
\begin{document}
\setdefaultlanguage{english}
\setotherlanguage[variant=ancient]{greek}
\end{lstlisting}

\end{frame}

\begin{frame}[fragile]	
	
Then, when using one of the `other' declared languages, toggle it when needed, either as
	
{\scriptsize \begin{lstlisting}
	\begin{greek}
		<greek text>
		...
	\end{greek}
\end{lstlisting}}
	
or, for single lines, as
	
{\scriptsize \begin{lstlisting}
	\textgreek{<greektext>}
\end{lstlisting}}

The former might look like this:

{\scriptsize \begin{greek}
ἐν ἀρχῇ ἦν ὁ λόγος, \\
καὶ ὁ λόγος ἦν πρὸς τὸν θεόν, \\
καὶ θεὸς ἦν ὁ λόγος. \\
\end{greek}}

\break

The latter might look like this:

{\scriptsize \textgreek{ὁ λόγος}}
