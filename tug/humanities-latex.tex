\begin{frame}
    \frametitle[Everything]{\LaTeX\ for Everything}
The world's best established software system for typesetting and document formatting, \LaTeX\ has been popular throughout the mathematical and physical sciences for decades. \break

The utility of \LaTeX\ for scholars in humanities disciplines, however, tends to be overlooked. \break

\LaTeX\ can be used for typesetting in most instances in which a scholar of humanities needs to present her materials. \break

I prepared these slides in \LaTeX\ -- with less frustration than would have been possible in graphical presentation program -- as 1 file:

\begin{frame}
	\frametitle[Tutorial]{This Tutorial}
In this tutorial, I will introduce attendees to the use of \LaTeX\ for a common task in humanities scholarship, namely, typesetting texts with unusual line length and other organizational needs, such as primary texts with our original translations. \break

The specific `thought experiment' will be that of determining how to prepare a handout for a historical conference or class presentation with perfectly organized excerpts of a Byzantine Greek text and an original English translation in parallel. \break
\end{frame}

\begin{frame}
	\frametitle[\LaTeX]{As a humanities scholar, why choose \LaTeX ? {\scriptsize (1/2)} }

In historical disciplines, editors expect us to prepare our materials for publications with bug-for-bug Microsoft Word compatibility for back-and-forth markup. In practice, this expectation requires use to use word processors, such as (my favorite option) LibreOffice Writer. \break

A \textbf{word processor} presents material in a \textbf{W}hat \textbf{Y}ou \textbf{S}ee \textbf{I}s \textbf{W}hat \textbf{Y}ou \textbf{G}et (\textbf{WYSIWYG}, ``whizzy-wig'') style. While one writes, one is simultaneously thinking about and working on formatting.

\end{frame}

\begin{frame}
	\frametitle[\LaTeX]{As a humanities scholar, why choose \LaTeX ? {\scriptsize (2/2)} }
	
In a \textbf{typesetting language}, such as \LaTeX\ (or \hologo{ConTeXt}), one is only concerned about the \emph{structure} of what one is writing. \break

One declares that specific characters will be in certain categories of writing `things' (heading, paragraph, footnote, accent type, typeface, etc.), and one allows the program to compile one's writing as `code' into a tidy final document -- every time. \break

Preparing a \textit{curriculum vitae}, conference handout, or other document with unusual typesetting requirements becomes about the writing itself -- not the formatting.
	
\end{frame}

\begin{frame}[fragile]
	\frametitle[Simple Parallel Text]{Simple Setting of Parallel Text in \LaTeX}
There are many ways to set parallel text in \LaTeX. \break

The \emph{simplest} is to use the community package called \texttt{parallel}, which is included in TeXLive, MiKTeX, and other \LaTeX\ distrubtions. For full documentation, visit: \break 

\href{https://ctan.org/pkg/parallel}{ctan.org/pkg/parallel} \break

To start using \texttt{parallel} in any \LaTeX\ document, simply include the following in the preamble: \break
 
\verb|\usepackage{parallel}| \break

\begin{frame}[fragile]
	\frametitle[Using parallel]{Using the \texttt{parallel} package}

Basic Format

{\scriptsize \begin{lstlisting}
\begin{Parallel}{}<left adjustment>}{<right adjustment>}
		\ParallelLText{<left text row 1>}
		\ParallelRText{<right text row 1>}
			\ParallelPar
		\ParallelLText{<left text row 2>}
	...	
\end{Parallel}	
\end{lstlisting}}

Example
	
{\scriptsize \begin{lstlisting}
\begin{Parallel}{0mm}{0mm}
	\ParallelLText{Ave Maria,}
	\ParallelRText{Hail Mary,}
		\ParallelPar
	\ParallelLText{gratia plena,}
	\ParallelRText{full of grace,}
\end{Parallel}	
		\end{lstlisting}}

\begin{Parallel}{0mm}{0mm}
\ParallelLText{Ave Maria,}
\ParallelRText{Hail Mary,}
\ParallelPar
\ParallelLText{gratia plena,}
\ParallelRText{full of grace,}
\end{Parallel}	

\end{frame}

\begin{frame}[fragile]
	\frametitle[Advanced Parallel Text]{Advanced Setting of Parallel Text in \LaTeX}
	
For \emph{advanced} scholarly setting of parallel text in \LaTeX, one ought to learn the 2 interconnected and frequently updated community packages \texttt{reledpar} (for parallel typesetting specifically) and \texttt{reledmac} (for typesetting academic editions in general), which are bundled with TeXLive and other \LaTeX\ distributions. \break

These 2 packages are specifically designed for long-form academic parallel texts, and they are meant to replace older methods of academic parallel text preparation in \LaTeX\ that you might find in Web searches. \break

For full documentation, visit: \break
\href{https://ctan.org/pkg/reledpar}{ctan.org/pkg/reledpar} \break
\href{https://ctan.org/pkg/reledmac}{ctan.org/pkg/reledmac} \break

Remember them in the preamble to your document:

{\scriptsize \begin{lstlisting}
\usepackage{reledpar}
\usepackage{reledmac}
\end{lstlisting}}

\end{frame}

\begin{frame}[fragile]
	\frametitle[Non-Latin Text]{Setting of Non-Latin Text in \LaTeX}

The most important package for handling text outside of Latin alphabets is the community package \texttt{polyglossia}, which is (as you probably have guessed) included in TeXLive and other \LaTeX\ distributions. \break

This fresh and continuously updated package is meant to replace older methods of non-Latin text entry that you might find in Web searches, notably \texttt{babel}, which ought to be set aside in 2026 (except in legacy projects that already rely on it). \break

For full documentation, visit: \break
\href{https://www.ctan.org/pkg/polyglossia}{ctan.org/pkg/polyglossia} \break

\end{frame}

\begin{frame}[fragile]
	\frametitle[Using polyglossia]{Starting a Project with \texttt{polyglossia}}

Remember the package in the preamble:

\begin{lstlisting}
\usepackage{polyglossia}
\end{lstlisting} 

\vspace{1em}

Then, \emph{after} the beginning of the document but \emph{before} actual type, declare the languages and their variants. For example, here is this .tex file's polyglossia declarations:

\begin{lstlisting}
\begin{document}
\setdefaultlanguage{english}
\setotherlanguage[variant=ancient]{greek}
\end{lstlisting}

\end{frame}

\begin{frame}[fragile]	
	
Then, when using one of the `other' declared languages, toggle it when needed, either for long excerpts as
	
{\scriptsize \begin{lstlisting}
	\begin{greek}
		<greek text>
		...
	\end{greek}
\end{lstlisting}}
	
or for single lines as
	
{\scriptsize \begin{lstlisting}
	\text<language>{<text in language>}
	e.g.
	\textgreek{<Greek text>}
\end{lstlisting}} \break

The former might be used to set apart long quotations, e.g. \break

{\scriptsize \begin{greek}
ἐν ἀρχῇ ἦν ὁ λόγος, \\
καὶ ὁ λόγος ἦν πρὸς τὸν θεόν, \\
καὶ θεὸς ἦν ὁ λόγος. \\
\end{greek}}

\break \vspace{1em}

while latter might be used to insert a keyword in another language, e.g. {\textgreek{ὁ λόγος}}, in the midst of prose.
	
\end{frame}

\begin{frame}[fragile]
	\frametitle[Combining parallel and polyglossia]{Combining \texttt{parallel} and \texttt{polyglossia}}

{\small The simplest way of showing multilingual text in parallel is} \break

{\scriptsize \begin{lstlisting}
\begin{Parallel}{0mm}{0mm}
	\text<language>{\ParallelLText{<ancient text 1>}}
	\ParallelRText{<modern text>}
		\ParallelPar
	\text<language>{\ParallelLText{<ancient text 2>}}
	\ParallelRText{<modern text 2>}
\end{Parallel}	
\end{lstlisting}} \break

\vspace{1em}

{\small but since this is tedious for more than a few lines, I remind everyone of \texttt{reledpar} and \texttt{reledmac} for longer and/or more complicated parallel text displays.} \break

\end{frame}

\textbf{BibLaTeX} -- comprehensive citation and bibliography management, capable of being interlinked with a variety of citation managers (Zotero, JabRef, etc.) \break

\textbf{TeXStudio} and other graphical front-ends -- LaTeX can be written in any plain text editor, but a designated LaTeX writing environment improves efficiency and helps correct code errors \break
